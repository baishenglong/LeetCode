## 题目

```
实现一种算法，找出单向链表中倒数第 k 个节点。返回该节点的值。

示例：

输入： 1->2->3->4->5 和 k = 2

输出： 4
```

## 题解

方法1：使用两个指针，`相距为k`

```python
# Definition for singly-linked list.
class ListNode:
    def __init__(self, x):
        self.val = x
        self.next = None

# Double pointer
class Solution:
    def kthToLast(self, head: ListNode, k: int) -> int:
        a = head
        b = head
        for i in range(k):
            b = b.next
        while b:
            a = a.next
            b = b.next
        return a.val

    
# Test
n1 = ListNode(1)
n2 = ListNode(2)
n3 = ListNode(3)
n4 = ListNode(4)
n5 = ListNode(5)

n1.next = n2
n2.next = n3
n3.next = n4
n4.next = n5

print(Solution().kthToLast(n1, 2))
```
