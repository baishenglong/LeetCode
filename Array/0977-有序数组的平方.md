## 题目

```None
给定一个按非递减顺序排序的整数数组 A，返回每个数字的平方组成的新数组，要求也按非递减顺序排序。

示例 1：
输入：[-4,-1,0,3,10]
输出：[0,1,9,16,100]

示例 2：
输入：[-7,-3,2,3,11]
输出：[4,9,9,49,121]
```

## 思路

- 利用map函数将列表每个元素平方
- 对平方后的列表进行排序

## 补充

- sorted函数可对所有可迭代对象进行排序，返回新的列表，不会对原来的数据做改动
- sort函数只能应用在列表上，对原列表进行排序，不会创建新列表

## 题解

```python
def sortedSquares(A):
    A = sorted(list(map(lambda x: pow(x, 2), A)))
    return A

print(sortedSquares([-4, -1, 0, 3, 10]))
```
